小样本学习
====
## 概念
基于少量样本生成好的模型<br>
核心问题：如何在有限的样本下，最小化经验风险


## 优势
1. 降低数据采集/标注成本<br>
2. 让机器智能更逼近人工智能<br>
3. 有可能降低算法任务的整体成本（研发工作量、工程部署、人力、时间等）

## 关键技术
1. 基于度量的思路：迁移学习，利用辅助数据集，学习样本之间相似度的度量方法，然后应用于小样本上；
2. 数据增强，也有部分专门针对小样本的数据增强
3. 基于预训练的迁移学习
4. 多任务学习

解决方案的分类：<br>
数据：利用先验知识，增强监督信号；（数据增强）<br>
模型：利用先验知识，缩小假设空间的大小<br>
参数求解：利用先验知识，更改假设空间中对最优假设的搜索
